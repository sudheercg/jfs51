Abstract classes:
----------------
-We cannot create an object of Abstract class.
-Objects can be created only for the child classes which inherits abstract class.  
- If a class has at least one abstract method, the class must be abstract. \
- abstract methods in abstract class must be implemented in the 
  child classes. 
- if we are  not implementing abstract method in the child class, then
   the child class must also be abstract class. 

- We can have a constructor for abstract class.
- Constructor is called  when child class object is created. 


Methods and Modifiers:
----------------------
Abstract methods cannot be private. 
They cannot be final
They cannot be static.


Abstract classes  are used when 
- we want a common behavior for some child  classes
- and some uncommon behavior for child classes.

- the methods with uncommon behavior are declared as abstract methods
- the methods with common behavior are declared as concrete methods..
